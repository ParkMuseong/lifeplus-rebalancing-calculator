# 포트폴리오 리밸런싱 도구

한국(KOSPI/KOSDAQ) + 미국 주식을 지원하는 Streamlit 기반 리밸런싱 계산기.

## 기능

1. **자동 종목 검색** — 종목명/티커 일부만 입력하면 코드와 현재가가 자동으로 채워집니다.
   - 한국: `FinanceDataReader`로 KOSPI/KOSDAQ 전체 종목 검색
   - 미국: `yfinance` 검색 + 티커 직접 입력 지원
2. **현재 비율 시각화** — 보유 수량을 입력하면 파이차트와 현재 vs 목표 비교 막대그래프로 즉시 표시
3. **리밸런싱 계산** — 목표 비율과 추가 입금액을 넣으면 종목별 **매수/매도 주식 수**를 정확히 계산
   - 모든 평가금액은 원화(KRW)로 환산 (USD/KRW 환율은 yfinance에서 자동 조회)
   - 수량 반올림 방식 선택 가능 (반올림/버림/올림)
4. **주문 가이드 출력** — 실제 증권사 앱에 입력할 매수/매도 주문 리스트 제공

## 실행

### 방법 1: 배치 파일 (가장 쉬움)

```
run.bat
```

가상환경 생성 → 패키지 설치 → Streamlit 실행을 한 번에 처리합니다. 최초 1회만 수 분 소요.

### 방법 2: 수동 실행

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

브라우저가 자동으로 열리며 `http://localhost:8501`에서도 접근 가능합니다.

## 사용 흐름

1. 상단 "종목 추가" 영역에서 한국/미국 탭을 골라 종목명을 입력 → 검색 결과에서 선택 → `추가`
2. "보유 수량 & 목표 비율" 표에서 각 종목의 보유 수량과 희망 비율(%) 입력
3. 목표 비율 합계가 100%가 되면 하단에 리밸런싱 결과가 자동 표시
4. 필요하면 "추가 입금액"을 입력해서 시나리오 비교
5. "주문 가이드" 섹션의 매수/매도 리스트대로 증권사 앱에서 주문

## 주의 사항

- 시세는 종가 기준이라 장중 실시간 가격과는 차이가 있을 수 있습니다.
- 미국 주식 검색은 yfinance 네트워크 응답에 따라 결과가 다를 수 있습니다. 정확한 티커를 입력하면 가장 확실합니다.
- 환율은 1시간 캐시, 시세는 1분 캐시입니다.
- **한국 종목 리스트**는 `data/krx_listing.csv`에 번들된 스냅샷을 사용해 즉시 검색됩니다. 신규 상장 종목이 누락되면 한국 주식 탭의 "종목 리스트가 오래된 것 같다면?" 영역에서 갱신할 수 있습니다. CSV 자체를 갱신하려면 `python scripts/refresh_krx_listing.py`를 로컬에서 실행 후 커밋하세요.
